# Data Structures and Algorithms

## Arrays
Arrays store elements in contiguous memory locations and allow fast access using indexes.

## Linked Lists
A Linked List stores elements as nodes connected through pointers. Types include:
- Singly Linked List
- Doubly Linked List
- Circular Linked List

## Searching Algorithms
- Linear Search: Checks elements one by one.
- Binary Search: Efficient searching on sorted arrays.

## Sorting Algorithms
- Bubble Sort
- Insertion Sort
- Merge Sort
- Quick Sort
- Heap Sort

Sorting arranges data in ascending or descending order and improves searching efficiency.
