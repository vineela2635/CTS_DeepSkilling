# Module 12 – Containerization Using Docker

## Overview

This module introduces the fundamentals of containerization using Docker. It covers Docker architecture, images, containers, Dockerfiles, Docker Compose, networking, volumes, and common Docker commands used to build, package, and run applications.

---

## Topics Covered

- Introduction to Containerization
- What is Docker?
- Docker Architecture
- Docker Engine
- Docker Images
- Docker Containers
- Docker Hub
- Dockerfile
- Docker Compose
- Docker Volumes
- Docker Networks
- Docker CLI Commands

---

## Learning Outcomes

After completing this module, I was able to:

- Understand the concept of containerization.
- Explain Docker architecture and its components.
- Differentiate between Docker Images and Containers.
- Understand the purpose of Dockerfiles.
- Learn Docker networking and volumes.
- Use basic Docker CLI commands.
