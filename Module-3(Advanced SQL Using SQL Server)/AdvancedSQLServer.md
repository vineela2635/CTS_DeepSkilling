# Advanced SQL Server

## Introduction

Advanced SQL Server concepts help in managing large-scale databases efficiently. These concepts improve query performance, data integrity, and reporting capabilities.

## Topics Covered

### Window Functions
- ROW_NUMBER()
- RANK()
- DENSE_RANK()
- OVER()
- PARTITION BY

### Common Table Expressions (CTE)
- Simple CTE
- Recursive CTE

### Views
Views are virtual tables created from SQL queries.

### Indexes
Indexes improve query performance by speeding up data retrieval.

### Transactions
Transactions ensure data consistency using ACID properties:
- Atomicity
- Consistency
- Isolation
- Durability

## Learning Outcome

- Implemented Window Functions.
- Used Common Table Expressions (CTE).
- Created Views and Indexes.
- Performed Transactions using COMMIT and ROLLBACK.
- Improved understanding of advanced SQL concepts.
